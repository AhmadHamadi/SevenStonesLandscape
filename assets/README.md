# Assets

Static files for the site.

## images/

Put image files here:

- **`hero.jpg`** — Hero section background (used in CSS).
- **`logo.png`** — Site logo used in the header and footer.

If your logo has a different filename, either rename it to `logo.png` or update the `src` in `partials/header.html` and `partials/footer.html`.
